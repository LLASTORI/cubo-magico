# Tags Fix — Launch Tag Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Aplicar automaticamente a tag `lançamento:NOME_DO_FUNIL|LAUNCH_TAG` ao contato CRM quando uma venda de lançamento ocorre via Hotmart webhook, e propagar `launch_tag` para `crm_contact_interactions` nas respostas de survey.

**Architecture:** Dois fixes independentes em edge functions. Gap 1: bloco não-bloqueante no `hotmart-webhook` após o automation engine, seguindo o padrão try/catch já existente. Gap 2: duas mudanças mínimas no `survey-webhook` — adicionar `launch_tag` ao select do funil e ao insert de interações. Nenhuma migration necessária — `crm_contacts.tags` (array) e `crm_contact_interactions.launch_tag` já existem.

**Tech Stack:** Deno, Supabase Edge Functions, TypeScript.

**Files:**

**Contexto crítico:**
- O arquivo tem ~2910 linhas. NÃO leia o arquivo inteiro — use offset/limit para ler apenas as seções relevantes.
- `contactId` é resolvido na linha 2306: `contactId = await findContactId(supabase, projectId, buyer?.email || null, buyerPhone);`
- O bloco de automation engine (linhas 2777–2829) é o modelo a seguir: try/catch não-bloqueante com `console.error` e sem re-throw.
- O novo bloco deve ser inserido **após o bloco de automation** (após a linha `console.error('[Hotmart Webhook] Error triggering automation:', automationError);` e o `}` de fechamento do catch) e **antes** dos `console.log('=== WEBHOOK PROCESSED SUCCESSFULLY ===')`.
- `purchase?.offer?.code` contém o offer_code da venda (mapeado a `offer_mappings.codigo_oferta`).
- `offer_mappings.provider` para Hotmart é `'hotmart'`.
- O padrão de tag é `lançamento:${funnel.name}|${funnel.launch_tag}` — usar acento em "lançamento".
- Só aplicar se `event === 'PURCHASE_APPROVED'` (não para refunds, chargebacks, etc.).

Localizar o trecho exato (após o `}` do catch do automation, antes de `console.log('=== WEBHOOK PROCESSED SUCCESSFULLY ===')`):

```typescript
    } catch (automationError) {
      // Don't fail the webhook if automation fails
      console.error('[Hotmart Webhook] Error triggering automation:', automationError);

    console.log('=== WEBHOOK PROCESSED SUCCESSFULLY ===');
```

Substituir por:

```typescript
    } catch (automationError) {
      // Don't fail the webhook if automation fails
      console.error('[Hotmart Webhook] Error triggering automation:', automationError);
    }

    // =====================================================
    // LAUNCH TAG — Apply lançamento tag to CRM contact
    // Non-blocking: failure must never abort the webhook
    // =====================================================
    try {
      if (contactId && event === 'PURCHASE_APPROVED') {
        const offerCode = purchase?.offer?.code;
        if (offerCode) {
          const { data: offerMapping } = await supabase
            .from('offer_mappings')
            .select('funnel_id')
            .eq('project_id', projectId)
            .eq('provider', 'hotmart')
            .eq('codigo_oferta', offerCode)
            .maybeSingle();

          if (offerMapping?.funnel_id) {
            const { data: funnel } = await supabase
              .from('funnels')
              .select('name, funnel_type, launch_tag')
              .eq('id', offerMapping.funnel_id)
              .single();

            if (funnel?.funnel_type === 'lancamento' && funnel?.launch_tag) {
              const launchTag = `lançamento:${funnel.name}|${funnel.launch_tag}`;

              const { data: contact } = await supabase
                .from('crm_contacts')
                .select('tags')
                .eq('id', contactId)
                .single();

              const existingTags: string[] = contact?.tags || [];
              if (!existingTags.includes(launchTag)) {
                const mergedTags = [...existingTags, launchTag];
                await supabase
                  .from('crm_contacts')
                  .update({ tags: mergedTags, updated_at: new Date().toISOString() })
                  .eq('id', contactId);
                console.log(`[LaunchTag] Applied tag "${launchTag}" to contact ${contactId}`);
              } else {
                console.log(`[LaunchTag] Tag "${launchTag}" already present on contact ${contactId}`);
              }
            }
          }
        }
      }
    } catch (launchTagError) {
      // Don't fail the webhook if launch tag logic fails
      console.error('[Hotmart Webhook] Error applying launch tag:', launchTagError);
    }

    console.log('=== WEBHOOK PROCESSED SUCCESSFULLY ===');
```
